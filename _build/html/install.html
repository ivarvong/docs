Installation Guide¶

The easiest way is to install newslynx is from the Python Package Index using pip or easy_install:

$ pip install newslynx

To install it manually simply download the repository from Github:

$ git clone git://github.com/newslynx/newslynx.git
$ cd newslynx
$ python setup.py install

Additional Dependencies¶

newslynx also depends on Postgres and Redis.
